<h2><a href="https://www.geeksforgeeks.org/problems/perfect-arrays4645/1?page=2&category=Arrays&difficulty=Basic&sortBy=submissions">Palindrome Array</a></h2><h3>Difficulty Level : Difficulty: Basic</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given an array <strong>arr[]</strong>, the task is to find whether the <strong>arr</strong> is&nbsp;</span><span style="font-size: 18px;">palindrome</span><span style="font-size: 18px;"> or not.&nbsp;</span><span style="font-size: 18px;">If the&nbsp;</span><strong style="font-size: 18px;">arr&nbsp;</strong><span style="font-size: 18px;">is palindrome then return</span><strong style="font-size: 18px;">&nbsp;true&nbsp;</strong><span style="font-size: 18px;">else return</span><strong style="font-size: 18px;"> false.</strong></p>
<p><span style="font-size: 18px;"><strong>Note: </strong>An array is said to be </span><span style="font-size: 18px;">palindrome</span><span style="font-size: 18px;"> if its reverse array matches the original array.&nbsp;</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>arr = [1, 2, 3, 2, 1]
<strong>Output:</strong> true
<strong>Explanation: </strong>If we reverse, we get [1, 2, 3, 2, 1] which is the same as before. So, the answer is true.
</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>arr = [1, 2, 3, 4, 5]
<strong>Output:</strong> false<br><strong style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">Explanation:  I</strong>f we reverse it, we get [5, 4, 3, 2, 1] which is the not same as before. So, the answer <span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">false</span><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">.</span></span></pre>
